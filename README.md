# ClojureScript template

This is a template that I like to use as a starting point for my Clojurescript projects.

## Dependencies

| Name         | Version |
|--------------|---------|
| shadow-cljs  | 2.22.1  |
| reagent      | 1.2.0   |

## Commands

You can run an application using `make` command. 
```shell
# Run development mode with
make watch 

# Build production version with
make release
```
